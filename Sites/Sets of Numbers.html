Sets of Numbers

Sets of numbers discussed in this chapter.

Numbers can be classified by certain properties they have and how they can be represented. Thus, we came up with sets of numbers to classify them. We will revisit this topic when we learn about algebra, so this will only be a brief introduction and overview.
You can think of sets as bags which have a bunch of numbers in them. We write sets with capital letters and their content contained by curly brackets. For example, a set could look like this: A = {1, 3, 5, 7, 9}. This set contains the numbers 1, 3, 5, 7 and 9. Just like with equations, we have symbols that denote relationships between numbers and sets. The "\(\in\)" symbol denotes membership, and can be read as "is an element of". So with our previous set we could write \(3 \in A\) (3 is an element of A), meaning that the number 3 is a part of the set A. Furthermore, \(\notin\) denotes non-membership, so we could say \( 4\notin A\) (4 is not an element of A).
A set doesn't have to be finite either, in fact the sets of numbers we will discuss in this chapter are all infinite. Let's introduce the first one:
\(\mathbb{Z}\) = {..., -3, -2, -1, 0, 1, 2, 3, ...} denotes the integers, that is all numbers that aren't decimals, both positive and negative. \(\mathbb{N}\) = {1, 2, 3, 4, 5, ...} denotes the natural numbers, which are just the positive integers, and mind you that zero is not a part of the natural numbers. We can write this with zero include as either \(\mathbb{N}_0\) or \(\mathbb{W}\) = {0, 1, 2, 3, ...} which are called the whole numbers. Notice that the whole numbers are larger than the natural numbers, by one number, that is 0, however smaller than the integers, which contain both the whole and natural numbers. Furthermore, we can describe almost every number using the rational numbers, denoted \(\mathbb{Q}\). The rational numbers describe every number that's a fraction, like \(\frac{1}{2}, \frac{7}{3}, \frac{7295731}{8572953} \text{ or } \frac{1}{1}\). You can't really describe this set by listing its elements as we did earlier, because previously we had a starting point, like 0 or 1 for example, however here there's nowhere we can really start. There is still a way to define this, however that will be discussed in a future chapter (see Set Builder Notation). The last set of numbers we'll discuss are the real numbers, denoted \(\mathbb{R}\). This denotes every number that can be represented algebraically. Technically there is another set which contains this set in itself as well, however that is also beyond the scope of this, see end of set builder notation.
There are certain numbers called irrational numbers, which can't be described with a fraction, but can be described in terms of mathematical operations, like \(\sqrt{2}\) for example. The proof for the irrationality of \(\sqrt{2}\) will come in a later chapter.
