Numeral Systems

A Numeral System is the way that numbers are written down using symbols. I will explain via examples.

Decimal System

The system we use is called the Decimal system, where every digit represents a power of 10 (deci meaning ten). Let's consider the number 1234 for example. The ones digits represents \(10^0\), so in our case \(4\cdot 10^0 = 4\). Next digit \(3 \cdot 10^1\), next: \(2\cdot 10^2, 1\cdot 10^3\). By this system every number can be represented as a sum of powers of tens.
Example: Write 78253 as a sum of powers of tens.
Solution: \(7 \cdot 10^4 + 8\cdot 10^3 + 2 \cdot 10^2 + 5 \cdot 10^1 + 3\cdot 10^0\)

Now that you understand the decimal system, let us discuss how number systems work generally. The decimal system is a number system with base 10. The base of a number system determines the base of every power that a digit represents.
